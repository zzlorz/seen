# @seen-tui/cli

Seen — a terminal UI for Git and code review.

Install

```
npm install -g @seen-tui/cli

# or run directly

npx @seen-tui/cli
```

Usage

```
seen
```

See the project README for full documentation.

Supported platforms
---

* macOS (arm64, x64)
* Linux (arm64, x64, glibc)
* Windows (x64)

The correct native binary is installed automatically via optionalDependencies.
